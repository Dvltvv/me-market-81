# MEmarket 8.1 - Хостинг

Мои жалкие кокетливые попытки хоста приложения для виндовс фон 8.1

## Структура

- `apps/` - .appx файлы приложений
- `api/` - API файлы для получения списка приложений
- `icons/` - иконки приложений
- `screenshots/` - скриншоты приложений
