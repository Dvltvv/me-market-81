# MEmarket 8.1 - Хостинг магазина приложений

Эта папка содержит файлы для хостинга магазина приложений MEmarket 8.1.

## Структура

- `apps/` - .appx файлы приложений
- `api/` - API файлы для получения списка приложений
- `icons/` - иконки приложений
- `screenshots/` - скриншоты приложений

## Использование

Приложение MEmarket 8.1 обращается к этим файлам для получения списка доступных приложений и их скачивания.

## Добавление новых приложений

1. Загрузите .appx файл в папку `apps/`
2. Добавьте информацию о приложении в `api/apps.json`
3. Загрузите иконку в папку `icons/`
4. Добавьте скриншот в папку `screenshots/`
